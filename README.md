# New Achievement — Community & Feedback

**[newachievement.ai](https://newachievement.ai)** is a fan-built site for readers of Matt Dinniman's *Dungeon Crawler Carl* series. Browse the quote library, upvote your favorites, submit lines from the books, and let the System AI roast you with achievements for your most pointless accomplishments.

This repository is the **public home for community input** — feature ideas, bug reports, quote corrections, and general discussion. The application itself is maintained separately; **you do not need to write code to participate here.**

---

## What you can do on the site

| Feature | What it does |
|---------|----------------|
| **Achievement generator** | Describe something you did. The System AI responds with a snarky achievement title, description, and useless reward — in character. |
| **Quote library** | Browse approved quotes with character art, book/chapter info, and filters by character or book. |
| **Upvotes** | Vote for quotes you love (one vote per quote per visitor). |
| **Quote submission** | Submit a quote and character name. An AI evaluator checks spelling, lore, and book/chapter, then approves, rejects, or flags for review. |

## How to use this repository

### Feature requests

Open an issue (label it **enhancement** or **feature request** if available) and describe:

- What you want to do or see
- Why it would improve the site for DCC fans
- Any examples (screenshots, quotes, mockups) if you have them

### Bug reports

If something on **[newachievement.ai](https://newachievement.ai)** is broken, open an issue (label it **bug** if available) with:

- What you expected
- What actually happened
- Browser/device (if relevant)
- Steps to reproduce

### Quote & content feedback

- **Missing or wrong quotes** — submit through the site first when possible; use an issue here if the submission flow itself is the problem.
- **Character list** — suggest characters to add to the filter/submission list (must be from the DCC universe).
- **Achievement tone** — share feedback if achievements feel off-character; include an example prompt and response if you can.

### General discussion

Use **GitHub Discussions** (if enabled) or issue comments for open-ended ideas: new sections, e-readers integrations, mobile UX, easter eggs, etc.
